# Alemeno Assignment

CoursesHub made using Reactjs(using Hooks) and Tailwind css.



### How build and run locally ?

Open terminal and type 

git clone https://github.com/suryansh-bhatnagar/alemeno-assignment.git

cd alemeno-assignment

npm install

npm start
